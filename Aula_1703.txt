Tema da aula: Script
    - São comandos executados (emperativamente) passo a passo
No Windons, um Script e um arquivo de texto contendo uma sequencia de comando que podem ser interpretados e execultados automaticamente pelo sistema operacional. Os scripts são usados para automatizar tarefas, configurar sistemas, manipular tarefas e execultar programas sem intervenção manipular
    - Os scripts no Windons podem ser escritos em diferentes linguagens, dependendo do interpretados usado
CMD = .bat
Powershell = .ps1
Visual Basic S. = .vsb

Paradgmas de Programação
    - Metodos ou metodologias de linguagens de Programação
- Imperativo (Python, C/C++, Visual basic, CMD, PS1)
    São linguagens que execultam linha por linha
    Sequencia de cima para baixo
Passo a passo dos comandos
- Orientação a Objetos (JavaScritp)
    Resumo, especificamente execultado por classe de objetos de forma não sequencial
    Tenta separar em classes, objetos, hierarquia
    Um outro estilo
- Funcional (Lrsp, Celulas do Excel)
Funções matematicas que não e nescessario que seja sequencial ou não sequencial
- Descritvo (HTML/CSS/XML)
